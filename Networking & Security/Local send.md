**LocalSend** is a **free, open-source file sharing application** that allows devices to transfer files **directly over a local network (LAN)**.

- No internet required
    
- No cloud storage
    
- No external server
    
- Cross-platform
    

It is designed as a **secure alternative to AirDrop** and similar tools.

## 3Ô∏è‚É£ How LocalSend Works

### Architecture:

1. Devices connect to the **same local network (Wi-Fi/LAN)**
    
2. LocalSend automatically **discovers nearby devices**
    
3. Uses **secure HTTP over local network**
    
4. Files are transferred **peer-to-peer (P2P)**
    

No cloud server involved.
