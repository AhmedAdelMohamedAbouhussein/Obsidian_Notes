## Overview

This repository contains my **comprehensive personal learning notes** organized using **Obsidian**. It covers topics ranging from **programming languages, computer science concepts, networking, web development, databases, machine learning, cloud computing, DevOps, system design, and security**. These notes are meant for study, reference, and revision purposes.

## Folder Structure & Topics

- **AI_ML/**: Artificial Intelligence, Machine Learning algorithms, Ray Tracing, Linear/Logistic Regression, KNN, SVM, GMM, types of ML.
    
- **Programming_Languages/**: Notes on Python, Java, C, C++, C#, Go, Rust, Zig, Kotlin, Swift, Ruby, Haskell, Elixir, MATLAB, COBOL, JavaScript, TypeScript, and more.
    
- **CS_Concepts/**: Data structures, algorithms, ACID, CAP Theorem, Sharding, Replication, Caching, Database Indexing, Isolation Levels, Memory Management.
    
- **Networking_Web/**: HTTP/1, HTTP/2, HTTP/3, 0-RTT, TLS versions, WebSockets, CDNs, DNS, REST, GraphQL, TCP/UDP, JS Runtime Engines.
    
- **Frontend_Development/**: CSS, JS frameworks, TUI, CLI, GUI, NUI, VUI, BUI, Hooks, Rasterization, React Diffing.
    
- **Backend_Cloud_DevOps/**: Node.js, Backend frameworks, AWS, DevOps, Package Managers, GitHub Actions, Microservices, Containers, Virtualization.
    
- **Systems_Linux/**: Linux fundamentals, Boot Process, Virtualization, Event Loop, Multithreading, Garbage Collection, System Concepts.
    
- **Security/**: Authentication, Authorization, Security best practices.
    
- **Side_Projects/**: Notes from side projects, experiments, and miscellaneous tech topics.
    
- **Tech_Terms/**: Definitions and quick-reference of important technology terms.
    
- **Assets/**: Images, diagrams, or other resources used in notes.
    

## Features

- **Markdown-based notes** compatible with Obsidian and GitHub.
