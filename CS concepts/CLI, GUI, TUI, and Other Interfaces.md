### 1. CLI — Command-Line Interface

**Meaning:**

- User interacts with the system by typing **commands** in a terminal.
- Input: keyboard
- Output: text in console

**Examples:**

- `git`, `ls`, `cd`, `python script.py`
- Terminals: Bash, PowerShell, Command Prompt

**Key Features:**

- Very fast for experienced users
- Lightweight, low resource usage
- Scriptable / automatable

**Memory Hook:** **“Type commands → system responds”**

**One-Line Exam Answer:**

> CLI is a text-based interface where users type commands to interact with the system.

---

### 2. GUI — Graphical User Interface

**Meaning:**

- User interacts via **windows, buttons, icons, and menus**.

**Examples:**

- Windows, macOS apps, web applications    

**Key Features:**

- Visual and intuitive
- Supports mouse + keyboard
- Heavier than CLI

**Memory Hook:** **“Click, drag, see graphics”**

**One-Line Exam Answer:**

> GUI is a visual interface where users interact using graphical elements like buttons, icons, and menus.

---

### 3. TUI — Text User Interface

**Meaning:**

- Text-based, but can include **windows, panels, colors, and menus** in a terminal
- More advanced than plain CLI    

**Examples:**

- `htop`, `vim`, `midnight commander`

**Memory Hook:** **“Graphical UI, but in text”**

**One-Line Exam Answer:**

> TUI is a text-based interface in a terminal that provides menus, windows, and forms for user interaction.

---

### 4. NUI — Natural User Interface

**Meaning:**

- Interface that responds to **natural human actions**: touch, gestures, voice, motion    

**Examples:**

- Touchscreens, Kinect, voice assistants like Alexa or Siri

**Memory Hook:** **“Interact naturally, no keyboard or mouse needed”**

**One-Line Exam Answer:**

> NUI allows interaction with a system using natural actions like gestures, touch, or voice.

---

### 5. BUI — Browser User Interface

**Meaning:**

- Interface presented via a **web browser**
- Often uses HTML, CSS, JavaScript    

**Examples:**

- Google Docs, Gmail, Facebook web

**Memory Hook:** **“Everything in a browser”**

**One-Line Exam Answer:**

> BUI is an interface accessed and used through a web browser.

---

### 6. VUI — Voice User Interface

**Meaning:**

- User interacts via **voice commands**
- Often overlaps with NUI

**Examples:**

- Siri, Alexa, Google Assistant    

**Memory Hook:** **“Talk to the computer”**

**One-Line Exam Answer:**

> VUI is an interface that allows users to control systems using voice commands.

---

### 7. Summary Table (All Interfaces)

| Interface | Input               | Output                   | Example              | Key Feature            |
| --------- | ------------------- | ------------------------ | -------------------- | ---------------------- |
| CLI       | Keyboard            | Text                     | `git`, Bash          | Fast, scriptable       |
| TUI       | Keyboard            | Text (with panels/menus) | `htop`, `vim`        | Advanced terminal UI   |
| GUI       | Mouse + Keyboard    | Graphics                 | Windows apps         | Intuitive visual UI    |
| NUI       | Touch/gesture/voice | Visual/Audio             | Touchscreens, Kinect | Natural interaction    |
