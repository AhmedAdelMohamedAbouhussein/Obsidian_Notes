**What it is:**

- Official Windows package manager by Microsoft.
    
- Installs, upgrades, and removes software from the command line.
    

**Key features:**

- Search for packages (`winget search <package>`).
    
- Install software (`winget install <package>`).
    
- Upgrade software (`winget upgrade <package>`).
    
- List installed apps (`winget list`).
