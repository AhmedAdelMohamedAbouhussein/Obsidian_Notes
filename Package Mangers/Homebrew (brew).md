**What it is:**

- Popular package manager for macOS (and now Linux & Windows via Homebrew for Windows).
    
- Installs command-line tools, apps, and libraries.
    

**Key features:**

- Simple syntax for installation, upgrading, and removal.
    
- Manages dependencies automatically.
    
- Supports macOS, Linux, and Windows.
