**What it is:**

- SDKMAN (Software Development Kit Manager) manages multiple SDKs (Java, Maven, Gradle, Kotlin, Scala, etc.).
    
- Useful for developers who need different SDK versions for different projects.
    

**Key features:**

- Install SDKs easily.
    
- Switch between versions globally or for a project.
    
- List all available versions of an SDK.
