**What it is:**

- Comes with Node.js.
    
- Installs **JavaScript packages/libraries** for Node.js projects.
    
- Used for front-end and back-end development with Node.js.
    

**Key features:**

- Install packages locally (for a project) or globally (system-wide).
    
- Manage dependencies listed in `package.json`.
    
- Run scripts defined in `package.json`.
